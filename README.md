# Aircnc

[Live Site](https://aircnc2.web.app/) | [Backend repository](https://github.com/sarowar-hosen1/Aircnc-server)

Aircnc is online hotel booking system.

<ul>
  <li>React.js</li>
  <li>React-router</li>
  <li>Redux</li>

  <li>ES6</li>
</ul>
